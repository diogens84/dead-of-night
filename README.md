# Dead of Night (2nd Edition) - Foundry VTT System

Implementación del juego de rol de horror **Dead of Night (2ª Edición)** para **Foundry Virtual Tabletop**. 

Un sistema enfocado en la gestión de recursos de supervivencia (Survival Points) contra la pista de tensión global (Global Tension Track).

---

## 📦 Instalación en Foundry VTT

Para instalar el sistema en Foundry VTT, ve a la pestaña **Game Systems** -> **Install System** y copia/pega el **Manifest URL** correspondiente:

### 🚀 Última Versión Estable (Recomendado)
```text
https://github.com/diogens84/dead-of-night/releases/latest/download/system.json
```

### 🏷️ Versión Específica (v1.1.5)
```text
https://github.com/diogens84/dead-of-night/releases/download/v1.1.5/system.json
```

---

## 🌟 Características
- **Hoja de Personaje**: Gestión de atributos, reducciones y tiradas.
- **Hoja de Monstruo**: Configuración de monstruos y amenazas.
- **Tensión Global & Puntos de Supervivencia**: Pistas automáticas integradas.
- **Soporte de Idiomas**: Español (`es`) e Inglés (`en`).
